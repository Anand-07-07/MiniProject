How it works

1

Capture your mood

Choose webcam, photo, or audio input to reveal the listener's emotional state.

2

Analyze emotion

The detector estimates one of seven emotions and computes confidence for stable suggestions.

3

Receive music

The recommender selects top songs and returns Spotify/YouTube links so you can listen immediately.

Setup guide

Install and run the app on Windows

Install dependencies

python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install --upgrade pip
pip install -r requirements.txt

Run modes

  • python -m src.cli webcam --duration 6
  • python -m src.cli image --image path\to\portrait.jpg
  • python -m src.cli audio --duration 5

Language & catalogue

Select Hindi or Marathi using --language hindi or --language marathi. Use --catalogue to load a custom JSON file.

Usage examples

Commands for every workflow

Webcam mode

Live mood detection with optional feed display.

python -m src.cli webcam --duration 6 --auto-play

Image mode

Detect emotion from a saved portrait image.

python -m src.cli image --image assets\portrait.jpg --language hindi

Audio mode

Infer mood from microphone recording and recommend a track.

python -m src.cli audio --duration 5 --language marathi --auto-play-platform youtube

Questions answered

Project details at a glance

Does it play music directly?

No. It opens Spotify or YouTube links in your browser so you can play the music there.

Can I add more songs?

Yes. Add entries to data/hindi_songs.json or use a custom JSON catalogue via --catalogue.

What if detection is low confidence?

Use better lighting for webcam mode, choose a clearer portrait, or adjust --min-confidence.
